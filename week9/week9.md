### 作业说明

作业 1：训练一个可以识别常见家居环境的家居机器人，对家居环境进行 grounding 和 counting，还能进行 VQA

- 数据参考使用
  [ https://cs.nyu.edu/~fergus/datasets/nyu_depth_v2.html](https://cs.nyu.edu/~fergus/datasets/nyu_depth_v2.html)
- 模型参考使用
  MM Grounding Dino + BLIP 2 或者 LLaVA-NeXT

作业 2：训练一个小型的 Coder 模型，跑通 Code Alpaca 项目

- 模式和数据使用 Code Alpaca，但不是从 0 到 1 训练，使用 LoRA/QLoRA 微调并对比前后效果即可；
  [ https://github.com/sahil280114/codealpaca](https://github.com/sahil280114/codealpaca)
